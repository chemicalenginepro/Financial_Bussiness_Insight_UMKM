# Business Insight AI 📊

Aplikasi analisis finansial otomatis berbasis Web App yang dirancang khusus untuk membantu UKM, Akuntan, Auditor, dan Investor dalam memetakan kesehatan keuangan bisnis secara instan.

## 🚀 Fitur Utama
- **Multi-Format Upload**: Mendukung file Excel (.xlsx), CSV, dan dokumen cetak PDF.
- **Auto Data Cleaning**: Menghapus baris kosong dan menyelaraskan format angka secara otomatis (Anti-Crash).
- **Interactive Dashboard**: Grafik tren keuangan responsive yang nyaman dilihat via HP.
- **Otak AI Gemini**: Analisis Rasio, Cash Flow, Prediksi Kebangkrutan, hingga Deteksi Anomali.
- **Professional Export**: Unduh hasil laporan dalam bentuk berkas PDF formal dalam satu klik.

## 🔑 Cara Penggunaan
1. Buka aplikasi via tautan Streamlit Cloud Anda.
2. Masukkan **Gemini API Key** gratis Anda (bisa didapatkan di Google AI Studio).
3. Unggah file laporan keuangan Anda.
4. Klik **Jalankan Analisis Komprehensif AI** dan tunggu laporan Anda selesai disusun.
5. Klik **Unduh Hasil Laporan PDF** untuk menyimpan file secara offline.
